初始化项目: npm init -y
安装hapi框架: yarn add hapi@16
env2安装: yarn add env2
hapi的swagger安装: yarn add hapi-swagger@7 inert@4 vision@4 package
参数验证joi: yarn add joi@13

- hapi 企业级开发node 框架
    - 后端开发中的一些plugins
        API文档 自动生成
- env2
    将私密的账号信息以配置文件的方式，加载进process中
    process.env
    .env放进.gitignore文件中实现不上传
